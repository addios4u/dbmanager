# Changelog

## [1.1.2] - 2026-04-09

### Added
- 테이블 뷰에서 **F5** 키로 데이터 새로고침 지원
  - 현재 정렬 상태 유지
  - WHERE 필터 입력창에 타이핑한 내용 자동 적용
  - 셀 편집·삭제 확인·행 추가 중에는 F5 무시 (데이터 손실 방지)
  - Refresh 버튼에 "(F5)" 힌트 표시
  - 한국어·일본어·중국어 간체 다국어 지원

## [1.1.1] - 2026-04-01

### Added
- 멀티 쿼리 일괄 실행 (Run All) 기능
- AI 쿼리 생성기 — Generate / Refine 모드

### Changed
- Explorer 사이드바 커넥션 드래그 앤 드롭 순서 변경 지원

### Fixed
- Open VSX 배포 지원으로 Cursor, VSCodium 호환성 확보

## [1.1.0] - 2026-03-15

### Added
- SSH 터널링 지원
- 데이터 Import (CSV, XLSX, JSON) 기능
- 데이터베이스 생성·삭제 커맨드

## [1.0.1] - 2026-03-01

### Fixed
- 버전 메타데이터 및 마켓플레이스 배너 적용

## [1.0.0] - 2026-02-20

### Added
- 최초 릴리스
- MySQL, MariaDB, PostgreSQL, SQLite, Redis 지원
- 테이블 데이터 뷰 · 편집 · 내보내기
- SQL 편집기 (Monaco Editor)
- Explorer 사이드바 트리뷰
